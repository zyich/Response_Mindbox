# Response_Mindbox

Вопрос №1.
### Граничные значения:

-1: Ошибка

0: Нельзя оформить

1: Нельзя оформить

14: Можно оформить

20: Можно оформить

45: Можно оформить

114: Можно оформить

115: Можно оформить

116: Ошибка

### Классы эквивалентности:

Недопустимые значения:
Диапазон: 
(−∞,0) и (115,+∞)
Примеры: -1, 116
Результат: "Ошибка"

Возраст, при котором нельзя оформить паспорт:
Диапазон: 
[0,14)
Примеры: 0, 5, 13
Результат: "Нельзя оформить"

Возраст, при котором можно оформить паспорт:
Диапазон: 
[14,115]
Примеры: 14, 21, 45, 115
Результат: "Можно оформить"
